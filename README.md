# Enterprise AI Knowledge Platform

## Overview

An enterprise-grade AI platform that enables organizations to search, understand, and retrieve knowledge from large collections of documents using modern Data Engineering and Generative AI technologies.

## Goal

Build a production-ready Knowledge Assistant using:

- Python
- PySpark
- Databricks
- Delta Lake
- AWS S3
- Vector Search
- LangChain
- FastAPI
